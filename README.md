# Claude Desktop Launcher Fix for Windows

A beginner-friendly Windows helper package for people who keep clicking Claude Desktop and accidentally opening Claude in the web browser instead.

This package is designed to help Windows users set up a cleaner Claude Desktop launch workflow and avoid confusion between Claude Desktop and Claude web.
