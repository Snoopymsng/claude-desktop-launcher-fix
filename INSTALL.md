# Install Notes

Thank you for buying **Claude Desktop Launcher Fix for Windows**.

This guide explains how to add the ZIP file as a Claude Desktop skill and how to use it safely.

## Before you start

Make sure you have:

* Windows 10 or Windows 11
* Claude Desktop installed
* The paid ZIP downloaded from Gumroad
* The file named:

```text
Claude Desktop Launcher Fix for Windows.zip
```

You do **not** need to extract the ZIP before adding it to Claude Desktop as a skill.

## How to add the skill to Claude Desktop

1. Open **Claude Desktop**.
2. Click your **profile icon** or **settings menu**.
3. Go to **Settings**.
4. Open **Customise**.
5. Find the **Skills** section.
6. Click **Add skill**.
7. Choose **Select ZIP**.
8. Select the downloaded file:

```text
Claude Desktop Launcher Fix for Windows.zip
```

9. Confirm the upload/import.
10. Wait for Claude Desktop to finish adding the skill.
11. Restart Claude Desktop if asked.

## After adding the skill

Once the skill has been added, start a new chat in Claude Desktop and ask:

```text
Use the Claude Desktop Launcher Fix for Windows skill and guide me through fixing my Windows Claude launcher.
```

Claude should then read the skill instructions and guide you through the correct steps.

## Optional admin script warning

This package may include an optional administrator script.

Only use the optional admin script if you understand what it does.

It may require administrator permission and may change how Claude web links behave on your Windows PC.

If you are unsure, do not run the admin script until you have read the notes inside the package.

## Basic troubleshooting

If Claude Desktop does not show the skill:

1. Close Claude Desktop fully.
2. Reopen Claude Desktop.
3. Go back to **Settings > Customise > Skills**.
4. Check whether the skill appears in the list.
5. If it does not appear, try adding the ZIP again.

If the ZIP will not upload:

1. Make sure you are selecting the ZIP file, not an extracted folder.
2. Make sure the filename ends in `.zip`.
3. Download the ZIP again from Gumroad if needed.
4. Restart Claude Desktop and try again.
