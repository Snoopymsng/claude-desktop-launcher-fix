# Install Notes

Thank you for buying Claude Desktop Launcher Fix for Windows.

## Before you start

Make sure you have:

- Windows 10 or Windows 11
- Claude Desktop installed
- The paid ZIP downloaded from Gumroad
- The ZIP extracted to a normal folder

## Basic install steps

1. Download the ZIP from Gumroad.
2. Right-click the ZIP file.
3. Click **Extract All**.
4. Open the extracted folder.
5. Read the included instructions before running any script.
6. Use the optional admin script only if you understand what it does.

## Safety note

Do not run PowerShell scripts from unknown sources without reading them first.

The optional admin script may require administrator permission and may change how Claude web links behave on your Windows PC.
